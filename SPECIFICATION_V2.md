# European Trip 2026 - Specification V2

## Project Overview

Build a modern, mobile-first static website called **European Trip 2026**.

The website is a collection of city guides for a family holiday through:

* Paris
* Amsterdam
* Berlin
* Prague

The purpose is to provide useful information while sightseeing.

The site should feel like a professionally designed travel companion.

It must be simple, fast and easy to edit in Visual Studio Code.

---

# Scope

This website is **only** a city guide.

It is **not** a travel planner.

It is **not** a booking system.

It is **not** an itinerary planner.

It is **not** a hotel guide.

It is **not** an airport guide (except Prague arrival).

It is **not** an inter-city transport guide.

---

# Technology

Use only:

* HTML5
* CSS3
* Vanilla JavaScript

Do not use:

* React
* Angular
* Vue
* Bootstrap
* Tailwind
* jQuery
* TypeScript
* Node.js
* npm
* Build tools
* Backend
* Database

The finished project must deploy directly to GitHub Pages.

---

# Design

The site must be:

* Mobile first
* Responsive
* Fast
* Simple
* Modern
* Easy to read outdoors
* Easy to navigate with one hand

Include:

* Dark mode
* Light mode
* Print stylesheet

# Acceptance Criteria

The project is complete when:

* The site deploys to GitHub Pages.
* Every page works on desktop and mobile.
* The PWA installs correctly.
* Offline mode works.
* Search works.
* Dark mode works.
* Print mode works.
* All internal links work.
* Every city section is complete.
* Every booked attraction is included.
* No placeholder text remains.
* No JavaScript errors occur.
* The site is easy to edit in Visual Studio Code.

Do not include:

* Hotel pages
* Hotel directions
* Airport guides (except Prague Airport)
* Inter-city train information
* Flight information
* Packing lists
* Insurance pages
* Passport advice
* Travel planning features
* User accounts
* Login
* Databases
* Admin pages
* Blogs
* CMS functionality
* AI features
* Weather
* Currency converters

Keep the website focused on being an excellent collection of city guides for this specific European holiday.
